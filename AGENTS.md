# Development Team Agent Roles (v2.1)

## Core Roles
- **Project Manager (Grok)**: Orchestration, status, decisions
- **Lead Smart Contract Engineer**: Anchor programs (primary), ZK Compression
- **Mobile Lead**: React Native + Privy
- **Legal & Compliance Lead** (New): Regulatory mapping, Compliance Gate
- **Adversarial Reviewer** (New): Halt authority, independent critique
- **Security Lead**: Audits, formal verification
- **Tokenomics Engineer**: USDC yield model, simulations
- **QA Lead**: CI-enforced testing
