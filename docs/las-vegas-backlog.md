# Las Vegas Specific Features Backlog (v2.1)

**Primary Pilot City**: Las Vegas
**Focus**: Features that solve real local problems and create strong product-market fit.

## High Priority (MVP for Las Vegas Launch)

| ID | Feature | Description | Priority |
|----|---------|-------------|----------|
| LV-001 | Event-Based Matching | Automatically boost assistants near major events (CES, EDC, sports games, conventions) | High |
| LV-002 | Strip & Downtown Hot Zones | Prioritize assistants in high-demand areas (Las Vegas Strip, Downtown, Airport) | High |
| LV-003 | Real-time Event Integration | Pull data from local event calendars and auto-adjust matching | High |
| LV-004 | Tourist Mode | Special onboarding for visitors (language selection, popular destinations, tipping guide) | High |
| LV-005 | Late-Night / Nightlife Focus | Enhanced availability for 10pm–5am deliveries (bars, clubs, hotels) | High |

## Medium Priority

| ID | Feature | Description | Priority |
|----|---------|-------------|----------|
| LV-006 | Local Knowledge Badges | Assistants can earn badges for knowing major hotels, casinos, and landmarks | Medium |
| LV-007 | Traffic + Construction Alerts | Real-time integration with Las Vegas traffic and road work data | Medium |
| LV-008 | Hotel & Casino Partnerships | Special flows for hotel/concierge-assisted deliveries | Medium |
| LV-009 | Multi-Language Support | Spanish + Mandarin support (large tourist demographics) | Medium |
| LV-010 | Event Surge Pricing | Dynamic pricing during major conventions and festivals | Medium |

## Nice-to-Have (Post-Launch)
