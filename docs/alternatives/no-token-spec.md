# No-Token Alternative Specification

**Goal**: Explore viability without $ASSIST token.

**Core Features**:
- USDC escrow only
- On-chain reputation (no staking yield)
- Platform fee 4-5% (all to ops + insurance)
- No governance token

**Pros**: Lower regulatory risk, simpler economics
**Cons**: No real yield for assistants, weaker network effects

**Recommendation**: Build this as parallel track in Phase 0.
